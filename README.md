# Profile CLI

CLI-приложение для управления профилями, которое позволяет создавать, просматривать, выводить список и удалять профили. Профили хранятся в YAML-файлах в текущей директории.

## Требования

- Go 1.19 или выше
- Git (для клонирования репозитория)

## Установка

### Клонирование репозитория

```bash
git clone https://github.com/yourusername/profile-cli.git
cd profile-cli
```

`mws` - это исполняемый файл (бинарник) нашего CLI-приложения, который создается после компиляции исходного кода. Название `mws` - это сокращение от **M**anager **W**ork **S**tation (менеджер рабочей станции).

- **В Linux/Mac:** `./mws`
- **В Windows:** `.\mws.exe` или `mws.exe`

Вы можете запускать приложение двумя способами:
1. **Скомпилированный файл:** `./mws` (быстрее, рекомендуется для использования)
2. **Через Go:** `go run main.go` (для разработки и тестирования)

Оба способа работают одинаково, но скомпилированная версия запускается быстрее.

## Быстрый старт
### 1. Через go run (без компиляции)
```bash
go run main.go <команда> [флаги]
```

### 2. Через компиляцию
```bash
# Компиляция (создает файл mws)
go build -o mws
```

# Создать профиль
./mws profile create --name=test --user=example --project=new-project

# Просмотреть профиль
./mws profile get --name=test

# Список всех профилей
./mws profile list

# Удалить профиль
./mws profile delete --name=test

# Справка
./mws help
